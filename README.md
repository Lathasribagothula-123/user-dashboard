# user Dashboard
A React-based User Dashboard application with user management features like search, filter, pagination, and CRUD operations.
# Features
- User List Display
- Search Functionality
- Filter by Status (Active / Inactive)
- Pagination
- Add User
- Edit User
- User Details Page
- Delete User
- Local Storage Support

 # Build using
- React (Functional Components + Hooks)
- React Router
- Axios
- CSS / Tailwind CSS
-
